docs

@get:Incremental
@get:InputFiles
@get:PathSensitive(value = PathSensitivity.RELATIVE)
abstract val docs: ConfigurableFileCollection

A file collection containing all the Markdown files included in this source set. These files are not generated, but rather describe the properties and code in the source set. They are checked to ensure that they are up-to-date with the source code and other configuration files. If the properties or code they're describing have changed, the markdown files are updated to reflect those changes.
