LazyMap

internal class LazyMap<K, V>(val compute: LazyMap<K, V>.(K) -> V)(source)

Creates a LazyMap which will invoke compute each time a new key is requested. Repeated lookups will not invoke the compute action again.

This is not a thread-safe map.

usage:

val cache = LazyMap<String, Data> { key ->
  // some expensive operation,
  // only performed when a key is requested for the first time
  someDataSource.get(key)
}
